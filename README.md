# RippleButton
  - RippleButton was created by adding ripple view which contain ripple layer. Users will feed smoothly touch inside button
  - See color in the button
  
### Installation
```sh
$ cd Example
$ pod install
```

### Tech
RippleButton uses technology:

* [Swift] - Language to write application for iOS ..etc
* [Cocoapods] - The Dependency Manager for iOS & Mac projects.
